<h2 id="retry-manager">Recovery standalone (RetryManager)</h2>

Si quieres manejar reintentos fuera del boundary:

```ts
import { RetryManager, createExponentialBackoff } from "react-rescuer/recovery";

const mgr = new RetryManager(
  {
    maxRetries: 3,
    retryDelay: (attempt) => Math.min(1000, 200 * 2 ** (attempt - 1)),
  },
  createExponentialBackoff(250, 10_000),
);

const { ok, attempt, delayMs } = mgr.next(
  "boundary-id",
  new Error("boom"),
  context,
);
```

Comportamiento

- `next()` incrementa el attempt por `boundaryId`.
- Si supera `maxRetries`, llama `onMaxRetriesReached` y retorna `{ ok: false }`.
- `reset(boundaryId)` limpia el contador.

<CalloutBox kind="info" title="Uso">
  Esto es util si quieres aplicar politicas de reintento fuera del
  ErrorBoundary.
</CalloutBox>
